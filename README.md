# functionalstudy
함수형 프로그래밍 스터디 그룹입니다.


### 연습용 환경세팅

설치

```shell
$ [sudo] npm i -g webpack
$ npm i
```

실행

```shell
$ npm start
```

URL : http://localhost:8080/
